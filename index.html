Сайт завода по изготовлению металлоконструкций

rsmetal.by

Сайт разработан с большего мной, достался мне уже с сверстаной немного шапкой, адаптивен. Все модальные окна и слайдер разработаны мной. Работа слайдера начинается с вызова функции с указанием желательных параметров, слайдер реагирует на поворот экрана на смартфоне и умееет рассчитывать свободное пространство с целью правильного размещения количества слайдов, настроена работа тач. Проект протестирован на последних версиях популярных браузеров.

Сайт ЖК «Некрасовский»

некрасовский.бел

Сайт адаптивен, разработан полностью мной. Есть возможность просмотра интересующей квартиры по нажатию на соответсвующий чертеж. На странице броней видны приобретенные квартиры. Бронирование квартир администратором осуществляется путем изменения значения ключа в объекте, всю остальную работу скрипт выполняет сам. Проект протестирован на последних версиях популярных браузеров.
